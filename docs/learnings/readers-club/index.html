Readers Club

Technical depth + human breadth. One book per month.

Format​

  • Month 1-2: Technical (CS, security, systems, math)
  • Month 3: Non-fiction (philosophy, psychology, history, biography)
  • Month 4: Wildcard (fiction, poetry, weird science)
  • Repeat

Current Cycle​

MonthBookTypeLead
JanCode: The Hidden Language (Petzold)Technical@miracleomokaro
FebThe Psychology of Money (Housel)Non-fiction@lead2
MarProject Hail Mary (Weir)Fiction@lead3
AprComputer Systems: A Programmer's Perspective (Bryant/O'Hallaron)Technical@lead4

Meeting Structure​

Week 1: Kickoff — context, author, why this book Week 2: Ch 1-3 discussion — concepts, disagreements, applications Week 3: Ch 4-6 — deep dive, connect to our work Week 4: Synthesis — what changed, what we ship, next book vote

Rules​

  1. Read before meeting — no spectators
  2. Bring quotes — page + insight + question
  3. No spoilers for fiction (separate channel)
  4. Ship something — code, writeup, diagram, talk inspired by book

Archive (Completed)​

  • Man's Search for Meaning (Frankl) — "Purpose survives suffering"
  • Atomic Habits (Clear) — "Systems > goals"
  • Zero to One (Thiel) — "Competition is for losers"
  • The C Programming Language (K&R) — "The source"

Join​

  • Discord #readers-club — vote on next book
  • GitHub resoluteraccoon/readers-club — notes, quotes, artifacts
  • Open to all members (Prospects: read-only)
