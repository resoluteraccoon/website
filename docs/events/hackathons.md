# Hackathons

We compete. We build tools under pressure. We break CTF challenges. We fix what we break. Win or lose, we debrief as a team.

## What We Do

- **CTF Competitions** — picoCTF, Hack The Box, TryHackMe events, university CTFs
- **Internal Hackathons** — 24-48 hour build sprints. Theme announced at start.
- **Tool Development** — Write exploits, scanners, automation scripts under time pressure.
- **Team Debriefs** — Post-event analysis. What worked. What failed. What we learned.
