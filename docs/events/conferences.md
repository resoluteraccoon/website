# Conferences

Black Hat, DEF CON, BSides — we attend as a crew. We split up, cover more ground, and reconvene to share what we learned.

## Target Conferences

| Conference | When | Focus |
|------------|------|-------|
| **Black Hat USA** | August | Cutting-edge research, briefings, trainings |
| **DEF CON** | August | Hacker culture, villages, CTFs, parties |
| **BSides** | Various | Community-driven, local chapters |
| **RSA Conference** | Spring | Industry, policy, vendor landscape |
| **ShmooCon** | Winter | Intimate, technical, East Coast |
| **Hack In The Box** | Various | Asia/Middle East focus, deep technical |

## How We Do It

1. **Divide & Conquer** — Each member picks 2-3 tracks. No overlap.
2. **Live Notes** — Shared doc, real-time summaries, key takeaways.
3. **Debrief Session** — Post-conference sync. Present top 3 findings each.
4. **Internal Webinars** — Deep-dive on most relevant talks for the crew.
