# Hacker Club

The Hacker Club is where theory meets the target. Live operations, capture-the-flag, vulnerability research, and exploit development — in a controlled, legal, high-stakes environment. No CTFd platforms. No hand-holding. Real infrastructure. Real vulnerabilities. Real growth. You don't play games. You operate.

## What We Do

### Movie Nights
Every Friday, we dissect a real breach. SolarWinds, Log4Shell, the Colonial Pipeline ransomware, the Vault 7 leak — we reverse-engineer the attack chain, the detection gaps, the response failures. No summaries. Primary sources only: CISA advisories, vendor advisories, incident post-mortems, leaked internal communications.

### Hackathons
Quarterly 48-hour operations. Custom infrastructure, real vulnerabilities, no scoreboard fluff.

- **Red Team** — Initial access → lateral movement → persistence → exfiltration
- **Blue Team** — Detection engineering → threat hunting → incident response → attribution
- **Purple Team** — Collaborative emulation, real-time feedback, joint improvement

Prizes aren't swag. They're mentorship sessions, introductions to hiring managers, and researchers, introductions to bug bounty programs.

### Conferences & Webinars
We attend as a crew. BSides, DEF CON, Black Hat, regional BSides — we coordinate travel, share hotel blocks, run our own private side events. We present our own research: exploit chains, detection rules, tooling we've built.

### Newsletter
**The Raccoon Report** — Weekly intelligence brief written by operators, for operators. No vendor pitches. No fluff. Active campaigns, new techniques, tool releases, vulnerability analyses, operational lessons from the field. Written in plain text. Encrypted in transit. Read in 5 minutes. Reference for months.
