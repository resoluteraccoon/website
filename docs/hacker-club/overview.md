# Hacker Club

The Hacker Club is where theory meets the target. Live operations, capture-the-flag, vulnerability research, and exploit development — in a controlled, legal, high-stakes environment. No CTFd platforms. No hand-holding. Real infrastructure. Real vulnerabilities. Real growth. You don't play games. You operate.

## What We Do

### Movie Nights
Every Friday, we dissect a real breach. SolarWinds, Log4Shell, the Colonial Pipeline ransomware, the Vault 7 leak — we reverse-engineer the attack chain, the detection gaps, the response failures. No summaries. Primary sources only: CISA advisories, vendor advisories, incident post-mortems, leaked internal communications.

### Hackathons
Quarterly 48-hour operations. Custom infrastructure, real vulnerabilities, no scoreboard fluff.

- **Red Team** — Initial access → lateral movement → persistence → exfiltration
- **Blue Team** — Detection engineering → threat hunting → incident response → attribution
- **Purple Team** — Collaborative emulation, real-time feedback, joint improvement

Prizes aren't swag. They're mentorship sessions, introductions to hiring managers and researchers, introductions to bug bounty programs.

### Conferences & Webinars
We attend as a crew. BSides, DEF CON, Black Hat, regional BSides — we coordinate travel, share hotel blocks, run our own private side events. We present our own research: exploit chains, detection rules, tooling we've built.

### Newsletter
**The Raccoon Report** — Weekly intelligence brief written by operators, for operators. No vendor pitches. No fluff. Active campaigns, new techniques, tool releases, vulnerability analyses, operational lessons from the field. Written in plain text. Encrypted in transit. Read in 5 minutes. Reference for months.

## How It Works

### Membership
You don't "join" the Hacker Club. You **earn your way in**.

1. **Complete the Foundations** — Linux Fundamentals, Networking Basics, Web App Basics in the Learning Hub (validated)
2. **Pass the Diagnostic** — 30-minute practical: Linux, networking, coding, debugging. No studying. Just shows where you are.
3. **Pick a Track** — Red, Blue, Purple, Research. Your learning pod forms around it.
4. **Show Up** — Movie Nights. Hackathons. Webinars. Newsletter contributions. The club runs on participation.

### The Standard
- **No spectators.** If you're in the room, you're on a team. You have a role. You deliver.
- **No anonymity.** Real names. Real faces. Real accountability. We hold each other to a standard because we respect each other enough to be honest.
- **No shortcuts.** You don't get the exploit until you understand the vulnerability. You don't get the detection rule until you've seen the attack.
- **Legal only.** Every target is authorized. Every scope is defined. Every action is logged. We build the skill to operate legally so we never need to operate otherwise.

### The Culture
> "Hey, you are over-talking."
> "Hey, you are beginning to say nonsense."

We have real conversations that shape us. We look each other in the eye and say the hard things. We hold each other accountable because we respect each other enough to be honest.

This is not a vague covenant group. This is not feel-good talk with no teeth. We have real conversations that shape us.

## For Visitors

You're welcome at Movie Nights and Webinars. No membership required. Just show up ready to engage. If you haven't read the source material, you'll listen more than speak — and that's fine. The discussions are technical, fast, and assume familiarity with the primary sources.

Hackathons and internal channels are members-only. Earn your way in.
