# Curated Learning

We don't do tutorials. We build operators.

## The Problem

Most "learning platforms" are content farms — SEO-optimized, shallow, designed to keep you watching, not building. You finish a course and still can't do the work.

## Our Approach

We curate the best free platforms in the world and structure them into **locked progression paths**:

### pwn.college
The gold standard for binary exploitation education. We sequence their modules from memory corruption fundamentals through advanced heap exploitation, with our own prerequisite checks and validation labs.

### TryHackMe
Selected rooms for network security, web application testing, blue team fundamentals. We filter out filler and chain rooms into coherent skill paths.

### PortSwigger Web Security Academy
The definitive resource for web application security. We organize their labs into progressive tracks: recon → injection → authentication → business logic → advanced topics.

## Locked Progression

You don't advance by watching videos. You advance by **proving competence**.

Each unit has a **validation challenge** — a practical exercise you must complete before the next module unlocks. No certificates for completion. No participation trophies. You demonstrate the skill, or you stay on the current module.

## Operator Debriefs

Every module ends with an operator debrief: what worked, what failed, what the real-world implications are. This is where theory meets steel.

## How to Start

1. **Pick a track** — Start where your gaps hurt most
2. **Check prerequisites** — Each module lists explicit dependencies. If you don't have them, do those first.
3. **Work through units** — Read, implement, verify. The validation challenge is the gate.
4. **Submit your artifact** — Code, writeup, teaching recording. A reviewer checks it against the rubric.
5. **Unlock the next node** — The Refinery updates your path. Downstream modules become available.

## No Solo Learning

You're assigned a **learning pod** (3–5 people). You sync weekly. You teach each other. Teaching is the final test of mastery — if you can't explain it to a peer, you don't own it yet.
