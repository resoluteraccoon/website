# Curated Learning

We don't do tutorials. We build operators.

## The Problem

Most "learning platforms" are content farms — SEO-optimized, shallow, and designed to keep you watching, not building. You finish a course and still can't do the work.

## Our Approach

We curate the best free platforms in the world and structure them into locked progression paths:

**pwn.college** — The gold standard for binary exploitation education. We sequence their modules from memory corruption fundamentals through advanced heap exploitation, with our own prerequisite checks and validation labs.

**TryHackMe** — Selected rooms for network security, web application testing, and blue team fundamentals. We filter out the filler and chain rooms into coherent skill paths.

**PortSwigger Web Security Academy** — The definitive resource for web application security. We organize their labs into progressive tracks: recon → injection → authentication → business logic → advanced topics.

## Locked Progression

You don't advance by watching videos. You advance by proving competence.

Each unit has a validation challenge — a practical exercise you must complete before the next module unlocks. No certificates for completion. No participation trophies. You demonstrate the skill, or you stay on the current module.

## Operator Debriefs

Every module ends with an operator debrief: what worked, what failed, what the real-world implications are. This is where theory meets steel.
