# Features Overview

Resolute Raccoon is built on five pillars: curated learning paths that eliminate noise, locked progression that forces mastery, AI-powered refinery that sharpens your code, thumbnail cards for rapid pattern recognition, and a progress tracker that measures what matters. No gamification. No streaks. Just measurable capability growth.

## The Five Pillars

### 1. [Curated Learning →](/docs/features/curated-learning)
We don't do tutorials. We build operators.

**The Problem:** Most "learning platforms" are content farms — SEO-optimized, shallow, designed to keep you watching, not building. You finish a course and still can't do the work.

**Our Approach:** We curate the best free platforms in the world and structure them into locked progression paths:
- **pwn.college** — Gold standard for binary exploitation. We sequence their modules from memory corruption fundamentals through advanced heap exploitation, with our own prerequisite checks and validation labs.
- **TryHackMe** — Selected rooms for network security, web app testing, blue team fundamentals. We filter out filler and chain rooms into coherent skill paths.
- **PortSwigger Web Security Academy** — Definitive resource for web application security. We organize their labs into progressive tracks: recon → injection → authentication → business logic → advanced topics.

**Locked Progression:** You don't advance by watching videos. You advance by proving competence. Each unit has a validation challenge — a practical exercise you must complete before the next module unlocks. No certificates for completion. No participation trophies. You demonstrate the skill, or you stay on the current module.

**Operator Debriefs:** Every module ends with an operator debrief: what worked, what failed, what the real-world implications are. This is where theory meets steel.

### 2. [Locked Progression →](/docs/features/locked-progression)
You don't skip steps. Locked Progression enforces prerequisite mastery before unlocking the next module. No jumping to exploit development without memory corruption fundamentals. No red teaming without OPSEC discipline. The gate isn't a quiz — it's a practical demonstration. Prove it, then proceed.

### 3. [Refinery AI →](/docs/features/refinery-ai)
Refinery AI is your personal code reviewer, exploit analyst, and architecture critic — trained on the world's best offensive and defensive codebases. It doesn't autocomplete. It challenges. It finds the off-by-one. It spots the race condition. It asks: "What happens when this fails?" Your code leaves the refinery harder than it entered.

### 4. [Thumbnail Cards →](/docs/features/thumbnail-cards)
Visual pattern recognition at speed. Thumbnail Cards distill complex exploits, protocols, and architectures into single-glance references: attack flow, trust boundaries, data transformations, and failure modes. Build a mental library of 500+ cards. See the pattern. Name the vulnerability. Move to the next.

### 5. [Progress Tracker →](/docs/features/progress-tracker)
What gets measured gets mastered. The Progress Tracker logs every lab completed, every exploit landed, every paper read, every hour trained — and visualizes your capability surface across 12 domains. No vanity metrics. No streaks. A living map of what you can actually do, backed by evidence.
