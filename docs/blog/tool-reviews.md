# Tool Reviews

Security tools. Productivity tools. Open source recommendations. Honest opinions.

## Recent Reviews

**Ghidra 11.x — The RE Platform** — Decompiler improvements, P-code analysis, headless analyzer, extension ecosystem. Where it beats IDA. Where it doesn't.

**Nuclei 3.x — Template-Driven Scanning** — Workflow syntax, code protocol, JavaScript execution, fuzzing module. Building custom templates for internal use.

**Sigma 1.x — Detection as Code** — Rule syntax, pipeline backend, Splunk/Elastic/OpenSearch output. False positive reduction strategies.

**Frida 16.x — Dynamic Instrumentation** — JavaScript API, Stalker, CO-RE, gadget injection. Hooking without root. Early instrumentation.

## Review Criteria

Every tool is evaluated on:

1. **Capability** — What does it actually do? What are the limits?
2. **Usability** — Learning curve. Documentation quality. Community support.
3. **Integration** — CI/CD pipeline compatibility. API availability. Output formats.
4. **Maintenance** — Release cadence. Issue response. Backward compatibility.
5. **Cost** — Not just money. Time to value. Operational overhead.

We do not review tools we have not used in production or real engagements. No press-release regurgitation.
