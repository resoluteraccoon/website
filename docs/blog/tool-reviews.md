## Review Criteria

- **Effectiveness** — Does it solve the problem?
- **Usability** — Learning curve, documentation, UX
- **Maintenance** — Active development, bug response
- **Community** — Plugins, templates, shared configs
- **Cost** — Free tier, licensing, value for money
