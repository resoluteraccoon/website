# About Resolute Raccoon

Resolute Raccoon is a blackhat-grade cybersecurity club built for those who refuse to compromise on skill, strategy, or standards. We operate at the intersection of offensive security, defensive mastery, and relentless self-improvement. No scripts. No shortcuts. Only the resolute.

## What We Do

- **Curated learning paths** — Hand-selected resources, sequenced for maximum skill transfer. No algorithmic recommendations, no SEO fluff.
- **Locked progression** — You don't unlock the next level by watching videos. You unlock it by proving competence. Every module gated behind demonstrated mastery.
- **Refinery AI** — Your personal adversarial coach. Hunts weaknesses, stress-tests assumptions, forces you to patch gaps before an attacker finds them.
- **Thumbnail cards** — Visual, information-dense references that distill complex topics into single-glance mental models. Build a library of 500+ patterns.
- **Progress tracker** — Concrete metrics: challenges completed, flags captured, papers read, hours in the lab. No vanity XP. Evidence-based growth.
- **Hacker Club operations** — Live CTFs, vulnerability research, exploit development on real infrastructure. Quarterly 48-hour hackathons. No hand-holding.

## Who Should Join

- **Operators** who want to sharpen every edge — offense, defense, code, hardware.
- **Builders** who write code that survives contact with adversaries.
- **Researchers** who read Project Zero writeups for fun and want to discuss them with peers.
- **Founders** building security products who need a network of elite practitioners.
- **Students** (university or self-taught) willing to outwork everyone else.

**Not for:** tourists, script kiddies, certificate collectors, or anyone looking for a Discord server to idle in.
